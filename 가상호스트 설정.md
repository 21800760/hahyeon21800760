# 계정추가 및 기본문서 추가

### 'choi' 계정을 만든다. 이때, adduser 명령어는 root 권한이 있어야 가능하다.
~~~
$adduser choi
~~~

* root 계정 암호 설정이 안되어 있다면 암호 설정하기!
~~~
$sudo passwd root
~~~

### $su choi -> html 폴더와 index.html파일 생성
~~~
$mkdir html
$vim index.html
~~~
